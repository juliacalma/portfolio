All-In-One — Case Study

A focused study companion: Pomodoro timers, categorised notes, and weekly insights.

Live / Store GitHub ← Back to Projects

Problem

Students juggle tasks, deadlines, and distractions. Existing tools are fragmented and not tailored to study routines.

Solution

One app that combines Pomodoro timers, categorized notes with images, and weekly reports to guide habits.

My Role

  • Business Analysis (requirements, user stories)
  • UX/UI wireframes & usability checks
  • Frontend dev (Flutter), Firebase integration

Stack

Flutter, Firebase Auth/Firestore/Storage, GitHub Actions (CI), Figma

Key Features

  • Pomodoro timer with alerts (light/dark mode)
  • Categorised notes with images and search
  • Weekly study insights & reminders

Outcomes

  • Beta: 120+ users, 78% weekly retention (4 weeks)
  • Avg. session: 18 mins; timer used 3×/day

Challenges & Learnings

Balancing feature scope vs. timeline; implementing offline-first notes; iterating based on user feedback.

Privacy

Only minimal analytics; no selling data; users can export/delete notes.
