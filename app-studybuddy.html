All-In-One — Case Study

A study companion app with focus timers and rich-text notes. I built the study timer and the Add Note page.

Flutter Dart Firebase Android Grade: A-

Back to Portfolio App Video GitHub

Problem

Students juggle tasks and distractions; existing tools are fragmented and don’t align with real study routines.

Solution

One app that combines a focus timer, rich-text notes (with images), and weekly insights to guide habits and reduce distraction.

My Role

  • Implemented the study timer with background detection and a 30s auto-pause + alert when the app is left.
  • Built the Add Note page using Flutter Quill (Zefyr) with bold/italic/underline and folder/category selection; wired persistence.

Tech & Approach

Flutter, Dart, Firebase (Auth/Firestore/Storage); Agile sprints with Git/GitHub and team QA.

Screenshots

Outcomes

  • Clearer routines via integrated timer + notes.
  • Consistent UI for quick capture and review.
  • Delivered to spec; graded A-.

Learnings

Balancing scope vs. timeline; handling background app state reliably; designing a simple notes UX that stays fast.
